# Personal Finance Dashboard

A simple Python application for managing personal finances and visualizing spending habits.

## Features

- Add Income
- Add Expenses
- View Current Balance
- Category-wise Expense Tracking
- Monthly Summary Report
- Expense Pie Chart Visualization

## Technologies Used

- Python 3
- Pandas
- Matplotlib

## Installation

```bash
pip install pandas matplotlib
```

## Run

```bash
python finance_dashboard.py
```

## Example

```text
1. Add Income
2. Add Expense
3. View Summary
4. Generate Chart
5. Exit
```
